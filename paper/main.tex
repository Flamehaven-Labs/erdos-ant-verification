\begin{abstract}
We present \texttt{erdos-ant-verification}, an open-source Python implementation
that computationally reproduces the finite, explicitly checkable parts of the
2026 OpenAI/Sawin disproof of the Erd\H{o}s planar unit-distance conjecture
\cite{OpenAI2026Proof,Sawin2026Remarks}. The central result is a numerical match
to equation~(2.2) of the remarks paper \cite{Sawin2026Remarks}: for the explicit
construction with $T = \{3, 5, 7, 11, 13, 17\}$, $S = \{101, \infty\}$, and
$L_T = \Q(\sqrt{5}, \sqrt{13}, \sqrt{17}, \sqrt{21}, \sqrt{33})$, we compute the
exponent excess to be $6.2391 \times 10^{-38}$, agreeing with the published
value $\approx 6.24 \times 10^{-38}$ to within $0.014\%$ relative error.

This artifact does \emph{not} contain a new mathematical proof, does \emph{not}
construct the infinite Golod--Shafarevich tower required for the asymptotic
result, and has \emph{not} been peer-reviewed. It is intended as a citable,
CI-verified reproducibility artifact for the published numerical lower bound,
not as an independent contribution to discrete geometry. The artifact is
available at
\url{https://github.com/Flamehaven-Labs/erdos-ant-verification}
under the MIT licence.

\noindent\textbf{Reproduction.}
\verb|pip install erdos-ant-verification && python -m erdos_ant.verify|
runs the unit tests and prints the verdict; the same command with
\verb|--write-evidence| also refreshes the tracked JSON certificate at
\path{reports/verification_result.json}. Pinned to commit
\texttt{<COMMIT\_HASH>}\footnote{Filled in at release-tag time; see
GitHub release \texttt{v0.1.2}.} and Zenodo
\texttt{<ZENODO\_DOI>}\footnote{For this artifact's first paper-bearing
release \texttt{v0.1.2}. The predecessor source-only release
\texttt{v0.1.1} carries DOI \href{https://doi.org/10.5281/zenodo.20366884}{10.5281/zenodo.20366884}.}.
\end{abstract}
